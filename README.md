# Converter-low-frequency-PWM-to-high-frequency-PWM-without-microcontroller.
Circuit converter low-frequency PWM to high-frequency PWM without microcontroller.

Input - 100Hz PWM 10%...95%;

Output- 32kHz PWM 20%...95%.
